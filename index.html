Hooman Benchmark
Dans ce projet, j'ai voulu refaire l'un des tests du site Human Benchmark que j'affectionne : le chimp test
Il s'agit d'un test dans lequel des boutons sont générés aléatoirement sur une grille et devant ensuite être sélectionnés dans l'ordre croissant.
J'ai également adapté proposé une version utilisable sur smartphone ce qui n'est pas le cas sur le modèle.
J'envisage également d'ajouter un modificateur de temps de réflexion dans le calcul du score.
Remake du Chimp test de Human Benchmark, avec prise en compte du format mobile.
CV generator
J'ai commencé par une page qui pouvait générer mon cv depuis un fichier excel, que j'ai ensuite fait évoluer pour utiliser une base de données MariaDB.
Cela m'a permis de découvrir des notions que j'abordais au travail sans y toucher sur des questions de scripts SQL et de gestion de base de données PHP.
Generateur de cv depuis une base de donnée MySQL en PHP.
